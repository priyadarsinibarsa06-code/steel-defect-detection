# Steel Defect Detection 🔍

## Overview
This project is an AI-based web application that detects defects in steel surfaces using image input.

## Features
- Upload steel image
- Detect defects (demo version)
- User-friendly interface

## Technologies Used
- Python
- Streamlit
- NumPy
- Pillow

## How to Run
1. Install requirements
2. Run: streamlit run app.py
